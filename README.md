# 🐾 TinDog

A fun, Tinder-inspired landing page for dogs — built with **HTML**, **CSS**, and **Bootstrap 5**.

![TinDog Preview](./images/iphone.png)

---

## 📖 About

TinDog is a fictional app landing page where dogs can find their perfect match. It showcases a modern, responsive web design with animated gradients, feature highlights, testimonials, and pricing plans.

> *"I no longer have to sniff other dogs for love. I've found the hottest Corgi on TinDog. Woof."*
> — Pebbles, New York

---

## ✨ Features

- **Animated gradient hero section** with app store download buttons
- **Feature highlights** — Easy to use, Elite Clientele, Guaranteed to work
- **Testimonial section** with press logos (TechCrunch, Mashable, Business Insider, TNW)
- **Pricing plans** — Chihuahua (Free), Labrador ($15/mo), Mastiff ($29/mo)
- **Responsive footer** with navigation links
- **Fully mobile-responsive** layout

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/tindog.git
   cd tindog
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` — no build step required.
